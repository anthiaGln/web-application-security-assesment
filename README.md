# Web Application Security Assessment 

This project is about a lab-based web application security assessment focused on identifying, understanding, and mitigating common web application vulnerabilities.

The project was conducted in a controlled and intentionally vulnerable environment for educational and learning purposes.

---

## Scope

The assessment covers the following vulnerability categories:

- Insecure File Upload Mechanisms  
- Remote Code Execution Risks  
- Local File Inclusion (LFI)  
- Remote File Inclusion (RFI)  

---

## Objectives

The main goals of this project are:

- To understand how common web application vulnerabilities occur
- To analyze their potential impact on a system
- To demonstrate secure configuration and mitigation strategies

---

## Assessment Environment

- **Application:** DVWA (Damn Vulnerable Web Application)  
- **Platform:** Metasploitable  
- **Environment:** Kali Linux
